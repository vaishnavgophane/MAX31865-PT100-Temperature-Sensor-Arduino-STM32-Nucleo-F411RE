# MAX31865-PT100-Temperature-Sensor-Arduino-STM32-Nucleo-F411RE
This repository contains a complete implementation of the MAX31865 RTD-to-Digital Converter interfaced with a PT100 sensor, using:  Arduino Uno / Mega  STM32 Nucleo-F411RE (SPI1 + UART2 Debug)  The project includes wiring diagrams, SPI communication code, and temperature conversion using the Callendar-Van Dusen equation.
